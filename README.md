# Netfilx-Movies-and-tv-shows-analysis
## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Jupyter Notebook

## Project Description
This project analyzes Netflix content data to identify trends related to
movies vs TV shows, release years, ratings, and country-wise distribution.

## Key Insights
- Netflix has more movies than TV shows
- Content production increased significantly after 2015
- United States contributes the highest number of titles
- Mature ratings dominate the platform
